```-a```: Muestra todos los sockets activos
```-n```: Muestra direcciones y puertos num√©ricos

```-u```: Filtra para mostrar solo sockets UDP
```-l```: Filtra para mostrar solo sockets en estado LISTEN
```-p```: Mostrar PID y  nombre del proceso
